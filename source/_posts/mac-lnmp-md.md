### 安装nginx
#### 通过brew下载nginx
1. 通过brew安装
2. 通过 brew 安装nginx
```
brew install nginx
```
#### 通过源码安装
1. 安装wget
```
brew install wget
```
2. 下载nginx源码包
```
wget http://nginx.org/download/nginx-1.8.0.tar.gz
```
3. ./configure
4. make
5. sudo make install
